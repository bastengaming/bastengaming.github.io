Tutorial: Cara pakai

  1. Jalankan server: python -m http.server 8080 di folder project.
  2. Buka PS4 > Browser > masukkan URL: http://IP-ADDRESS-PC:8080
  3. Pilih firmware > jalankan GoldHEN. Jika sudah sukses akan muncul popup.
  4. Untuk offline cache: setelah pertama kali berhasil, tekan tombol "Make Offline" jika tersedia, lalu matikan internet dan coba akses lagi.
